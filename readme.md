## Installation

```bash
# Clone this repository
$ git clone https://github.com/facalz/bloons-bot

# Go into the repository
$ cd bloons-bot

# Install dependencies
$ yarn install

# Don't forget to configure your .env file

# Run the app
$ yarn start or node .
```
